About us

Auth-Wiki is a library authentication application that allow users to create different authentication code samples and interact. Users can easily access/download our code samples or able to view example usage.

The users can make use of the library code when creating an authentication code or when browsing through the library.

At AuthWiki, guests are not also left out as they can easily view, interact with the documentation and register to contribute.

Though, one of the cons of being a guest is the inability to download code while on the other hand, the registered users have full access, can interact easily, make contributions, able to view example usage and download  code samples.
